다음 명령을 실행하고 출력 결과에 대해 설명 하시오.

sleep 100 &
sleep 100 &
// 첫 번째, 두 번째 sleep 명령 백그라운드에서 실행

jobs
// jobs를 통하여 백그라운드 작업 리스트 출력 

ps –f
// 프로세스 목록을 자세히 보여줌 (-f)을 통해
// 프로세스 ID(PID), 부모 프로세스 ID(PPID), 명령어 등을 포함한 정보 출력 

kill %1
// 작업 번호 1번에 해당하는 백그라운드 작업 종료
// sleep 100 (번) 종료 

ps -f
// 프로세스 목록 확인시
// 작업 번호 1번의 프로세스가 사라지고, 작업 번호 2번인 두 번째 sleep만 남게됨.
